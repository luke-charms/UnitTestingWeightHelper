A simple testing practice, where both manually written unit tests are used, as well as automatic generated test suites (generated by Randoop).
The WeightHelper.java file is the main part of code being tested. It contains two functions, one for calculating your BMI using height and weight, and another for calculating your BMI class label.
The manual test suite made sure to test all happy cases, as well as edge cases for full coverage.
In addition, a automated test suite generation was used too, using Pytest and Randoop as the main tools.
